# Email Builder

This is the Hobby Next.js project that allows you to build email templates focused on the Email Builder and Sending Email.

![Demo](./public/demo.png)

## Tech Stack

- Next.js
- Tailwind CSS
- Shadcn UI
- Convex (Just tested it)
- Nodemailer

## Convex

```bash
npx convex dev
```

## Run the project

```bash
npm run dev
```
